Stateless by Design

VeriBot stores no messages, logs no users, and builds no profiles. Every request is handled in complete isolation — the architecture enforces privacy, not just a policy.

Zero message logging — nothing is persisted to disk
SQLite cache stores URL hashes only — never content
1-hour TTL — cache entries expire automatically
No user identity stored — stateless per message
